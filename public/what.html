A map + a path.

Most people get lost because they learn fragments. This project connects the fragments into a system you can navigate.

The Civilization Graph (Map)

The graph shows how occupations connect to tools, technologies, industries, and the computing pillars that are digitizing the world.

Open it here: docs/graph.html

The Engineering Tree (Path)

The tree is a structured growth path: foundations → branches → projects → proof → opportunity.

If you want direction after exploring the graph, start with the tree and a beginner project track.

How You Use It

Use this formula to convert information into outcomes:

Field → Role → Skills → Tools → Projects → Income

  • Search your role (or target role) in the graph.
  • Pin it. Note tools/tech and computing pillars nearby.
  • Pick one project track and ship proof weekly.

Data + Scope

The web build is O*NET-powered and focuses on occupations, industries, tools/tech, and related roles. Heavier layers (skills/knowledge/abilities/tasks) can be added later.

This is a navigation tool, not personal career advice. Verify decisions with multiple sources.
